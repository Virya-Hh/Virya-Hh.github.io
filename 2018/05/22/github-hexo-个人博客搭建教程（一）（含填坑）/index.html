github+hexo 个人博客搭建教程（一）（含填坑）

  • github
  • hexo
  • 环境配置

前期准备

  1. Windows7-64bit
    查看方式：计算机右击—>属性—>查看系统类型

  1. 在github官网注册账号



创建代码库：选右上角的“+”–>New repository–>只需要填写Repository name: yourname.github.io(yourname自己设置即可,此处有坑（后缀格式必须为github.io，否则容易出错)

–>creating repository–>点击Settings–>GitHub Pages中选择change theme,之后即可生成，如下图所示：


  1. 下载Nodejs
  2. 下载Git
  3. 检查安装是否成功： 计算机–>搜索程序与文件–>输入cmd，在弹出的黑色界面中分别输入：
    node -v
    npm -v
    git –version


我选择的版本应该是比较新的,上述命令的输出分别为为：node： v8.10.0 npm:5.6.0 git:2.16.2

安装hexo

  1. 选择一个路径（我选择的是D:/），并且新建一个文件夹命名为hexo(其他名称也OK)
  2. cmd–>输入 cd D://hexo进入hexo的根目录下
  3. 然后输入 npm install hexo-cil -g(此处有坑：需要在联网状态下在线安装，不然肯定出错)
  4. 继续输入 npm install hexo –save
  5. 最后测试是否安装成功：输入hexo -v，若输出为hexo:3.7.1，即为安装成功。

注意：在进行上述的第3个步骤时，出现了错误（含ERR!或者FATAL!，若含WARN不是出错，可以忽略..),显示为：npm ERR! code ENOTFOUND…

解决办法:这里提供了三种解决方法，我只尝试了第一种，尝试成功了！

将Hexo与gitHub联系起来

  1. 配置Git个人信息 cmd输入–>进入hexo根目录下输入：

git config –global user.name “yourname”(yourname就是你刚才设置的)
git config –global user.email “your name@163.com“ （此处是输入你的邮箱，不一定是163邮箱）
–>生成密钥（继续输入以下）
ssh-keygen -t rsa -C “yourname@163.com“

注意：在进行上述的生成密钥时，出现了错误，显示为：Permission denied (publickey).
fatal: Could not read from remote repository.
Please make sure you have the correct access rights and the repository exists.

解决办法：https://www.jianshu.com/p/9b67641b5ec7 这是我试过的所有教程里讲的最详细且有效的！

  1. 配置Deployment
    打开hexo的根目录（D：\hexo）找到_config.yml–>进行以下设置：
    deploy:
    type: git
    repo: git@github.com:yourname/yourname.github.io.git
    branch: master

注意：每一个冒号后边有一个空格，很容易忽略的细节，也是很容易掉进去的坑。

  1. 生成页面

hexo new post “article title”

hexo s -g

hexo d -g

几个概念
在一开始装的时候，容易搞不清楚的地方：
首先需要在cmd中输入：hexo new post “article title”（表示新建一篇文章，默认存储在D:\hexo\source_posts中，对文章进行编辑就是在这个路径下打开你的文章，使用markdown进行编辑（我暂时在用这个，如有推荐的可以互相交流））接下来有两个步骤，可能会让你糊涂：

hexo s -g (这个指得是生成了一个本地浏览的网址，https://localhost:4000，是指的当你对你的博客进行了修改之后，这个可以通过刷新迅速的看到)

hexo d -g(这个指得是把你最终需要上传到你博客上的修改进行部署，这个步骤之后你就可以在你的博客主页上看到变化https://yourname.github.io/）
你看到的一些其他的博客上对主题进行修改时会出现两个名词：

站点配置文件：指的是D:\hexo_config.yml

主题配置文件：指的是D:\hexo\themes\主题名称_config.yml
