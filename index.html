Network论文笔记

Pyramid Stereo Matching Network
code

Abstract

  1. Depth estimation can be formulated as a supervised learning task (with CNNs)
  2. Question: 当前基于图块匹配的Siamese网络的架构，缺少利用环境信息取寻找不适定区域（遮挡区域，弱纹理区域等）对应关系的能力
  3. Solution: propose PSM Net, a pyramid stereo matching network

  4. PSMNet网络的有两个模块：金字塔池化和3D卷积神经网络。

    金字塔池化模块通过聚合不同尺度和不同位置的环境信息构建匹配代价值（cost volume）来充分利用全局环境信息。3D CNN通过将多个堆叠的沙漏网络（hourglass network）与中间监督(intermediate supervision)结合起来，去调整匹配代价值。

java基础课程实践回顾

JAK1.5 特性–>静态导入

静态导入的作用：简化书写

静态导入可以作用一个类的所有静态成员

静态导入的格式：

import static 包名.类名.静态的成员；

静态导入需要注意的是想：

  1. 如果静态导入的成员和本类的成员存在同名的情况下，那么默认使用本类的静态成员。如果需要指定使用静态导入的成员，那么需要再静态成员前面加上类名。

JAK1.5 特性–>增强for循环

增强for循环的作用：简化迭代器的书写格式（底层还是迭代器遍历）

增强for循环的使用范围：如果实现了Iterable接口的对象或者数组对象都可以使用for循环

增强for循环的格式：
for(数据类型 变量名：遍历的目标){
}

增强for循环需要注意的事项：

  1. 增强for循环底层也是使用了迭代器获取的，只不过获取迭代器有jvm
    完成，不需要我们获取迭代器而已，所以在使用增强for循环元素的过程中不准使用集合对象对集合的元素个数进行修改。

  2. 迭代器遍历元素与增强for循环变量的区别：使用迭代器遍历集合的元素时可以删除集合的元素，而增强for循环变量集合的元素时，不能调用迭代器的remove方法删除元素。

  3. 普通for循环与增强for循环的区别：普通for循环可以没有变量的目标，而增强for循环一定要有变量的目标。

Hive 编程指南学习

第一章 基础知识：

MapReduce： 将计算任务分割成多个处理单元然后分散到一群家用的或服务器级别的硬件机器上
HDSF: 分布式文件系统

Question：如何将数据基本框架（传统数据）转移到Hadoop上？
基本数据架构是基于传统关系数据库和结构化查询语句（SQL）

Solutions:Hive提供了一个Hive语言查询的SQL方言，Hive可以将大多数的查询转换为MapReduce任务（job）

Hive不是一个完整的数据库，Hadoop以及HDFS的设计本身约束和局限性限制了Hive所能胜任的工作。

Hive的最大限制： Hive不支持记录级别的更新、插入或者删除操作。但是可以通过查询生成新表或者将查询结果导入到文件中

Hive不支持事务

MapReduce操作会将集合中的元素从一种形式转换为另一种形式。在这种情况下，输入的键-值对会被转换成零到多个键-值对输出。其中输入和输出的键必须是完全不同，而输入和输出的值则可能完全不同。

某个键的所有键值对会被分到同一个reduce操作中。reduce过程的目的是将值得集合转换成一个值（对一组值求和或者求平均），或者转换为另一个集合。这个reducer最终会产生一个键值对。如果job不需要reduce的话，也可以没有reduce这个过程。

Hadoop分布式文件系统，管理者集群中的数据。每个数据块（block）都会被冗余（默认冗余3份），保证不会因为单个硬盘或者服务器的损导致数据丢失。

相机成像性能测试

References:

  1. EMVA 2018
  2. EMVA1288中文翻译
  3. EMVA 1288 概述：成像性能
  4. 如何评估相机灵敏度
  5. camera —(1) 摄像头图像质量常用指标的测试方法

基本的相机规格：帧率、分辨率和接口等
相机的成像性能： 量子效率、暗噪声、饱和度

量子效率（简称QE）:

计量单位：百分比 (%)

定义：传感器以特定波长将光子转换为电子的百分比。

这实际意味着什么？量子效率高的传感器意味着该传感器在将入射光线转换为电信号时效率更高。QE 越高意味着探测光线的灵敏度更高；这点对大多数应用环境而言都特别有用，对微光应用环境而言更是如此。此外，某些传感器可能还经过特别调整，以便在不同的波长范围内实现更高的灵敏度。根据您的应用环境，特定波长下的特定 QE 结果可能比其他因素更为重要。比如，在近红外范围 (850-950nm) 内，QE 越高越有利于微光交通监测。FLIR 出售的某些最佳传感器的峰值 QE 高达 70%-80% （参见图像，查看图表示例）。没有任何传感器的效率能够达到 100%。
