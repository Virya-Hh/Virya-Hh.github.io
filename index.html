2018-05-24-绑定域名出现的问题及解决

Question： 404：There isn’t a GitHub Pages site here.If you’re trying to publish one, read the full documentation to learn how to set up GitHub Pages for your repository, organization, or user account.

Tinking:

1. 域名状态：注册宽限期（addPeriod）
2. DNS非万网DNS，检查是否有影响
3. 域名与github关联是否有问题
4. Ping+域名，每次查询IP地址都会发生变化，原因？

Solution:

  1. 参考域名状态解析

  2. 本质上说说没有影响的，只是选择了不同的DNS服务器进行解析，但是用其他的DNS服务器，需要自己去检测时候可以解析，而使用万网的DNS服务器，会自动选择可以解析的服务器，因此最好选用万网DNS。此处的坑：修改DNS之后会提醒你去查看whios的DNS信息，若不一致，建议修改为一致的。这里需要时间等待，一般情况下修改完之后隔一段时间whios的会进行更新，不要一点开看到不一样就又修改为原来的DNS，这里浪费了很长时间。

  3. 参考 GitHub Pages 绑定来自阿里云的域名

2018-05-24-软件测试了解及数据结构练习

软件测试工程师在线题库及考试试题

腾讯测试工程师笔试

  1. 黑白盒测试的概念性问题
    黑盒测试 也称功能测试或者数据驱动测试或规格说明的测试，是从用户观点出发的测试。不考虑内部结构，只知道输入与输出之间的关系或者程序功能需求的情况下，来确定测试用例和推断测试结果的正确性和可操作性。黑盒测试方法主要有等价类划分法、边界值分析法、因果图法、错误推测法、综合策略、正交分析法，用于软件的确认测试。
    白盒测试也称逻辑驱动测试或者结构测试或基于结构的测试。它依赖于对程序细节的严密检验，针对特定条件和循环集设计测试用例，对软件的逻辑路径进行测试。在程序不同点检验程序的状态，来判断真实情况是否和预期的状态相一致。白盒测试方法主要包括逻辑覆盖发法，主要包括语句覆盖、判断覆盖、条件覆盖、判断条件覆盖、条件组合覆盖、路径覆盖

  2. 提高软件质量和可靠的技术大致可分为两大类：其中一类就是避开错误技术，但避开错误技术无法做到完美无缺和觉悟错误，这就需要软件设计能够容错能力强。

  3. 软件设计包括体系结构设计、接口设计、数据设计和过程设计。
